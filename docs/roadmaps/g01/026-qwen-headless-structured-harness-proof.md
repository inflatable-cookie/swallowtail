## Bounded First Proof

One exact Qwen Code `v0.19.11` executable performs one read-only structured run
against one exact configured access and model route. The driver sends content
through stdin, consumes bounded stream JSON, selects safe mode and approval
posture explicitly, excludes mutating, shell, network, background, and
subagent behavior, and applies exact native wall/tool/turn bounds.

The route is `AmbientHost`. Qwen's container-backed `--sandbox` is neither
required nor enabled. Durable project-scoped local transcript retention is
explicitly accepted; Swallowtail exposes no resume or deletion claim. The
host deadline and cancellation remain authoritative and join the process.

Card 080 freezes exact tool names and output schemas from the pinned source;
card 081 maps that corpus into the production driver. If a later stable-
artifact delta cannot suppress ambient customizations or mutating tools
without enabling a container, stop and re-contract the route.
